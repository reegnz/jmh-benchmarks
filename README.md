JMH benchmarks
==============

To run the benchmarks, you could do one of the following

The gradle run approach
-----------------------
```
./gradlew run
```

The gradle install approach
---------------------------
```
./gradlew install
build/install/jmh-benchmarks/bin/jmh-benchmarks
```

The tar.gz approach
-------------------
```
./gradlew build

```

Take the tar.gz in build
